# W3C OS — Live Demo

**An AI-native operating system built on W3C standards.**

TypeScript + DOM/CSS compiled to native binaries. No browser. No V8.

This Space runs the W3C OS showcase application as a native binary on a virtual display,
streamed to your browser via noVNC.

**What you're seeing**: A real native window rendered by tiny-skia (CPU), laid out by Taffy (Flexbox/Grid),
running as a 2 MB binary — not a web page.

## Controls

- **Click** on buttons and UI elements
- **Scroll** with mouse wheel
- **Tab** to navigate between focusable elements
